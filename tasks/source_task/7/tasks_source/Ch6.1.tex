%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\subsection{Работа. Мощность}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Трактор перемещает платформу со скоростью $7.2 \,\text{км/ч}$, развивая тяговое усилие в $25 \,\text{кН}$. Какую работу совершит трактор за $10 \,\text{мин}$?
}
%
{$A = F v t = 30 \,\text{МДж}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Со дна реки глубиной $4 \,\text{м}$ поднимают камень объёмом $0.6 \,\text{м}^3$ на поверхность. Плотность камня $2500 \,\text{кг/м}^3$. Найти работу по подъёму камня.
}
%
{$A = (\rho_{к} - \rho_{в}) g V h = 36 \,\text{кДж}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Поршень двигателя перемещается на $20 \,\text{см}$ под давлением $800 \,\text{кПа}$. Определите работу, совершаемую двигателем за один ход поршня, если площадь поршня $150 \,\text{см}^2$.
}
%
{$A = l P S = 2.4 \,\text{кДж}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Трактор равномерно тянет плуг, прилагая силу в $10 \,\text{кН}$. За $10 \,\text{мин}$ он проходит путь $1.2 \,\text{км}$. Определить мощность, развиваемую трактором.
}
%
{$P = \frac{F l}{t} = 20 \,\text{кВт}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Легковой автомобиль, развивая силу тяги $700 \,\text{Н}$, движется со средней скоростью $72 \,\text{км/ч}$ в течение одного часа. Какую работу при этом совершает двигатель автомобиля?
}
%
{$A = F v t = 50400 \,\text{Дж}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Человек равномерно толкает вагонетку массой $0.7 \,\text{т}$ по горизонтальному участку пути длиной $200 \,\text{м}$. Какую работу совершает человек, если сила трения составляет $0.06$ силы тяжести вагонетки?
}
%
{$A = 84 \,\text{кДж}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Вычислите мощность насоса, подающего ежеминутно $1300 \,\text{л}$ воды на высоту $24 \,\text{м}$.
}
%
{$P = \frac{m g h}{t} = 5.2 \,\text{кВт}$}
