\subsection{Равноускоренное движение}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Поезд, трогаясь с места, через $t_1 = 10\,\text{с}$ приобретает скорость $v_1 = 0.6\,\text{м/с}$. За какое время от начала движения скорость поезда станет равной $v_2 = 3\,\text{м/с}$? Движение поезда считать равноускоренным.}
%
{$t = 50\,\text{с}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Первоначально покоившееся тело начинает двигаться с постоянным ускорением $a = 3\,\text{м/с}^2$. Определить путь, пройденный телом за $t = 0.1\,\text{ч}$ после начала движения.}
%
{$s = \frac{a t^2}{2} = 194.4\,\text{км}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Подъезжая к светофору со скоростью $v = 10\,\text{м/с}$, автомобиль тормозит в течение времени $t = 4\,\text{с}$ и останавливается рядом со светофором. На каком расстоянии от светофора находился автомобиль в начале торможения?}
%
{$20\,\text{м}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Автомобиль с хорошими шинами может иметь ускорение $a = 0.5\,\text{м/с}^2$. Какое время потребуется для разгона автомобиля до скорости $v = 60\,\text{км/ч}$? Каков путь разгона в этом случае?}
%
{$278\,\text{м}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Межпланетная автоматическая станция «Марс-1» начала свой полет со скоростью $v_0 = 12\,\text{км/с}$. Благодаря притяжению Земли в конце первого миллиона километров её скорость уменьшилась до $v = 3\,\text{км/с}$. Считая движение равнозамедленным, найти ускорение полёта.}
%
{$-0.0675\,\text{м/с}^2$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Самолёт пробегает по бетонированной дорожке расстояние $s = 790\,\text{м}$. При отрыве от земли его скорость $v = 240\,\text{км/ч}$. Какое время продолжался разбег, с каким ускорением двигался самолёт?}
%
{$a = 2.81\,\text{м/с}^2, \quad t = 23.7\,\text{с}$}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\z{Тело, двигаясь прямолинейно с ускорением $a = 2\,\text{м/с}^2$, за время $t = 0.1\,\text{мин}$ прошло путь $s = 42\,\text{м}$. Какой была начальная скорость тела $v_0$?}
%
{$v = 1\,\text{м/с}$}
