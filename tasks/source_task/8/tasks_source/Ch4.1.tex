 \z{Какой минимальной высоты нужно взять зеркало, чтобы человек высотой $H$ смог увидеть себя во весь рост?}
%
{$\frac{H}{2}$}

%
{$\tan\alpha = n$}

{$25\degree$ или $65\degree$}
